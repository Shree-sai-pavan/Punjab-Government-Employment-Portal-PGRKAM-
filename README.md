IF YOU WANT RUN LOCALLY IN YOUR COMPUTER PLEASE FOLLOW THE STEPS:

Step-by-Step Guide
1.Download the ZIP file from the GitHub repository by navigating to the main page of the repository, clicking the green Code button, and selecting Download ZIP.
2.Extract the contents of the downloaded ZIP file to a new, specific folder on your local machine using your operating system's file explorer (e.g., right-click and select "Extract All").
3.Open VS Code on your computer.
4.In VS Code, go to the top menu and select File > Open Folder....
5.Navigate to and select the folder where you extracted the project files.
6.Click "Select Folder" to open the project in your VS Code workspace.
